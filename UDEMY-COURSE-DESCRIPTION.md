### üéØ What We're Building

You'll build a complete **real-time order processing system** that:

- **Producer Service**: Generates orders and sends them to Kafka every 2 seconds
- **Fraud Detection Service**: Analyzes orders in real-time to detect suspicious transactions
- **Payment Service**: Processes payments asynchronously
- **Analytics Service**: Stores orders in Couchbase and calculates running totals
- **Backend API**: Serves data via FastAPI REST endpoints
- **Frontend Dashboard**: Displays live order analytics in a React dashboard
- **AWS Deployment**: Everything running on AWS ECS with auto-scaling and load balancing

**Live Demo:** See the system in action at [orders.jumptotech.net](https://orders.jumptotech.net)
